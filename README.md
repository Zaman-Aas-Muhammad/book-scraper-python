# Book Scraper

A Python web scraper that extracts book data
from books.toscrape.com across multiple pages.

## Features
- Scrapes Name, Price, Rating
- Saves data to Excel (.xlsx) format
- Covers 31 pages automatically (500+ books)

## Technologies Used
- Python
- BeautifulSoup
- Requests

## How to Run
pip install requests beautifulsoup4
python scraper.py
